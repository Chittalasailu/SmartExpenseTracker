# 💰 Smart Expense Tracker

A full-stack expense tracking application that helps users manage their daily expenses, visualize spending patterns, and generate reports.

---

## 🚀 Features

- ➕ Add Expenses
- ✏️ Edit Expenses
- ❌ Delete Expenses
- 🔍 Search Expenses
- 🏷️ Category Filter
- 📊 Dashboard Analytics
- 🥧 Category-wise Pie Chart
- 📈 Monthly Expense Bar Chart
- 📄 Export Expenses to PDF
- 💻 Responsive User Interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Chittalasailu/SmartExpenseTracker.git
```

### Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```

Backend runs on:

```
http://127.0.0.1:8000
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
